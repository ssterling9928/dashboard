## 🚀 About This Project

A complete customized dashboard for my personal homelab, featuring server logs, metrics, docker services, and more.  All from an easy to launch webpage using synology backend APIs.

## 🛠️ Tech Stack

This project was built using a modern, performance-focused tech stack:

| Technology | Purpose |
| :--- | :--- |
| **React** | Core framework for webpage UI. |
| **Tailwind CSS** | Utility-first CSS for rapid and consistent styling. |
| **TypeScript** | For type safety and backend API routes. |
